# Wildlife 3D Cinematic Gallery

An immersive 3D wildlife gallery built with HTML, CSS, JavaScript, and GSAP.

## Features
- 3D rotating slider
- Cinematic detail pages
- Interactive animations
- Responsive design
- Glassmorphism effects

## Technologies
- HTML5
- CSS3
- JavaScript
- GSAP

## Run Locally
1. Open in VS Code
2. Use Live Server
3. Enjoy the experience
